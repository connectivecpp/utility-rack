Detailed Description

Overloaded utility function and class, primarily used with std::visit.

See https://en.cppreference.com/w/cpp/utility/variant/visit for explanations and usage. This is the basic and simple version of the code as copied and pasted from the cppreference site. There is likely to be a more elaborate std::overload in a future C++ standard (probably post-C++20 - see http://www.open-std.org/jtc1/sc22/wg21/docs/papers/2018/p0051r3.pdf).
